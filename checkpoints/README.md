# ALiDE

This is the code repo for **ALiDE: Adaptive Model Lightweighting and Retraining for Accurate Video Analysis in Diverse Edge Environments**.
We will provide a revised report after the manuscript is accepted.

## Description

This repository contains the code and resources used in the article "ALiDE: Adaptive Model Lightweighting and Retraining for Accurate Video Analysis in Diverse Edge Environments.". The code demonstrates how to build and train a simple demo of our system using PyTorch and OpenCV.
